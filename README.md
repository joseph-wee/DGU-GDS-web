
# 동국대학교 졸업 판별 웹 어플리케이션

# 개요
- 공학인증 및 복잡한 졸업조건으로 인한 학생 및 학과사무실의 시간소요를 해결하기 위해 시작한 프로젝트입니다.
- 관리자가 졸업조건을 입력하고 학생들의 정보 파일을 업로드하면 졸업 판별 결과를 얻을 수 있습니다.

# 프로젝트 일정
- 기획: 23.03 ~ 23.12.
